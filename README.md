# PSFilterPdn

PSFilterPdn is a [Paint.NET](http://www.getpaint.net) effect plugin that enables the use of 3<sup>rd</sup> party Adobe® Photoshop®-compatible filters.

 ## How to install the plugin

1. Exit Paint.NET.
2. Place PSFilterPdn.dll and PSFilterShim.exe in the Paint.NET Effects folder (usually C:\Program Files\Paint.NET\Effects).
3. Restart Paint.NET.
4. The plug-in will now be available as the 8bf Filter menu item in the Paint.NET Effects menu.

## Installing Photoshop-compatible filters

See [Installing 8bf filters](https://github.com/0xC0000054/PSFilterPdn/wiki/Installing-8bf-filters)
