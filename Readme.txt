Photoshop-compatible filter host Effect plugin for Paint.NET

Description
------------

This plugin allows Paint.NET to run 3rd-party Adobe Photoshop-compatible filters.

For more details, or to report bugs, please refer to the website:
D:\Program Files\paint.net\Effects
  http://psfilterpdn.codeplex.com/

Installation
-------------

1. Exit Paint.NET.
2. Place PSFilterPdn.dll and PSFilterShim.exe in the Paint.NET Effects folder (usually C:\Program Files\Paint.NET\Effects).
3. Restart Paint.NET.

A copy of Paint.NET can be obtained from the official website.
  http://www.getpaint.net/
